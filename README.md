# Cryptocurrency News Tracker

A Node.js application for tracking and aggregating cryptocurrency news from various sources.

## Features

- Fetch latest cryptocurrency news
- Track specific coins (BTC, ETH, etc.)
- Display trending crypto topics
- Simple and easy-to-use API

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Development

```bash
npm run dev
```

## Configuration

Create a `.env` file in the root directory:

```
API_KEY=your_api_key_here
```

## Technologies

- Node.js
- Axios for HTTP requests
- dotenv for environment variables
